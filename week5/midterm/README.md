## Project Name:
Simple Bank System

## Author:
Lunpei Yu

## Description:
This is a simple banking system written in C++.
It allows users to create accounts, deposit, withdraw money, 
transfer funds, and view transaction history and generate account statements

## Features:
- Create new bank account
- Deposit money into account
- Withdraw money from account
- Transfer money between account
- View account information
- View transaction history
- Generate account statements

## Concepts Used:
- Program Structure: Organized using functions, function prototypes, 
clean indentation and comments, compiles and runs without errors
- Input/Output : cin, cout, getline()
- Variables & Data Types: int, double, string, bool
- Control Flow: if, switch
- Loops: for, while, do-while
- Functions: multiple user-defined functions with parameters and return values
- String Manipulation: find(), substr(), concatenation
- Arrays: store accounts and transactions
- Struct: Account and Transaction

## How to use:
1. Run the program
2. Choose an option from the menu
3. Follow the prompts to perform different banking operations 
4. View results directly in console
